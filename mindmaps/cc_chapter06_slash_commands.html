Essential Slash Commands — The Pro's Toolkit
Master these and you'll operate at a completely different level
/model
Switch between Haiku, Sonnet, Opus. History sent to new model on switch. No tokens wasted.
/effort
Set reasoning depth: Low → Max. High effort means deeper agentic loop, more planning time per step.
/cost or /usage
See exact token spend this session and weekly. Current session %, week %, days until reset.
/context
Shows live breakdown: system prompt, tools, skills, messages, free space. Your RAM usage dashboard.
/compact
Summarizes the entire conversation into a digest. Frees up context. Like "minutes of meeting" for Claude.
/rewind
Time-travel back to any previous chat state — reverting both conversation AND all file changes made since.
💡
⭐ PRO WORKFLOW
Start sessions with Opus + High effort for planning. Switch to Sonnet for building. Use Haiku for maintenance. Check /cost and /context regularly to stay efficient.

Rewind — Your Time Machine
The most underused power feature in Claude Code
→
What it does — takes you back to any previous chat message AND reverts all file changes made since that point
→
Like a save state — exactly like dying in a video game and respawning from your last checkpoint
★
When to use — Claude went in the wrong direction, renamed a file you didn't want renamed, created wrong structure
→
How to use — type /rewind, select the checkpoint message to return to, confirm. All file changes since that point undo.
✓
Safe to use — reverted files stay accessible via git history. Rewind is always recoverable.
⏪ Rewind💾 Checkpoint✅ Always safe

/context — Your Usage Dashboard
See exactly what's eating your context window
→
System Prompt — ~6k tokens: Claude Code's core instructions on how to behave and operate
→
System Tools — ~8k tokens: definitions of every tool Claude has (Read, Write, Bash, etc.)
→
MCP Tools — varies: each connected MCP server sends its tool list. 10 MCPs = massive overhead
→
Skills — minimal: only loaded when invoked, not always present
→
Messages — grows: your conversation history, biggest target for /compact
★
Free space — what's left. When this gets low, quality drops and costs spike. Watch this number.
