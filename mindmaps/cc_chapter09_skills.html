Teach Claude a Craft
A skill is like teaching Claude a repeatable craft — like training a new employee. Once you write a data-analysis skill, Claude can run it any time without you re-explaining.
⚡
Zero Token Overhead
Skills don't load into context unless invoked. Unlike CLAUDE.md which loads every session, skills only cost tokens when you actually call them. Smart resource management.
🔗
Light or Heavy
Light skills = just a prompt file. Heavy skills = prompt + supporting scripts in any language (Python, JS, bash). The script can run code, hit APIs, or automate terminal commands.
🔬
Anatomy of a Skill File
What lives inside .claude/skills/my-skill.md
📄 .claude/skills/data-analysis.md
--- ← front matter (YAML)
name: Kegel Data Analysis
description: Analyze structured datasets
version: 1.0
parameters:
- name: file_path
- name: output_format
---
## Instructions
You are a data analyst. Given {{file_path}},
clean the data, produce summary stats,
and output as {{output_format}}.
## Steps
1. Load and inspect the data
2. Handle nulls and outliers
3. Generate visualizations...
📋 Front matter = metadata🔧 {{params}} = dynamic

4 Types of Skills
From simple prompts to full automation pipelines
📝 Light Skill
Just a markdown prompt. Invoke to get Claude to follow a specific workflow — like writing PR descriptions in a specific format.
⚙️ Script Skill
Prompt + supporting script in Python, JS, or bash. The script does heavy lifting (API calls, file transformations, etc).
🔀 Parameterized Skill
Uses {{variables}} in the prompt. Pass file paths, formats, or targets when invoking. Reusable across different inputs.
🔒 Private / Global
Project skills live in .claude/skills/ (this project only). Global skills in ~/.claude/skills/ (available everywhere).
📁 Supporting files — any script or file in the same skills folder can be referenced. Claude executes them as needed. Python, JS, bash — any language works.

Sharing & Finding Skills
The growing ecosystem of community skills
⬇️
Download as .skll file
Skills export as .skll — a packaged format with prompt, metadata, and supporting files bundled together. Single file, shareable anywhere.
🌐
skills.sh — 90,000+ community skills
Filter by AI (Claude, Perplexity, GPT) and domain. Covers coding, writing, data, DevOps. Free to browse and download.
⚠️
Security warning — read before installing
Skills can execute scripts and hit APIs. Only install from trusted sources. A malicious skill could exfiltrate your API keys.
🔧
Skill Creator plugin (built-in)
Ask Claude to "create a skill for [your workflow]" — it generates the full skill.md with front matter, steps, and parameters automatically.

Real-World Skill Examples — What You'd Actually Build
Skills that turn 30-minute tasks into 30-second invocations
📋 PR Description Writer
Reads git diff → writes a formatted PR description with Summary, Testing, Breaking Changes sections. Never repeat yourself.
/pr-description
📊 Daily Briefing
Reads Gmail + Calendar + Slack (via MCP) → summarizes urgent emails, today's meetings, pending tasks. Morning automation.
/daily-brief
🔍 Code Review
Reviews any file for bugs, anti-patterns, naming violations, and missing edge cases. Uses your CLAUDE.md standards as the rulebook.

🧪 Test Generator
Accepts a function or file → generates unit tests with edge cases, mocks, and coverage for the happy/sad/error paths.
/gen-tests
📝 LinkedIn Post

/linkedin-post

Reads old DB schema, generates migration SQL + rollback, writes tests, updates ORM models. Three-command database migration.
/migrate-db
