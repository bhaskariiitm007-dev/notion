What "Agentic" Actually Means
The loop that separates an AI agent from a chatbot
1
Plan — Claude reads your request, then creates a step-by-step internal plan before doing anything
2
Act — picks a tool and executes: creates a file, runs bash, reads a file, fetches the web
3
Observe — reads the output of that action: was it correct? Did it fail? What happened?
4
Adapt — if there's an error, it fixes it and tries again. If it worked, moves to the next step.
5
Complete — only reports "done" when it has verified the output actually works
⭐
THE AGENTIC INSIGHT
Unlike a chatbot that answers in one shot, Claude Code loops through actions — thinking, doing, observing, correcting — until the task is genuinely complete.

Token Cost Reality Check
Understanding pricing to avoid surprise bills
$
Token = 3/4 of a word — every word in and out costs tokens. Both input AND output are charged.
$
Sonnet pricing — $3 per million input tokens, $15 per million output tokens. 1M ≈ 750,000 words.
$
Haiku is cheapest — $1/million input, $5/million output. Use for simple tasks to save significantly.
$
Opus is premium — $15/million input, $75/million output. Reserve for planning and complex reasoning only.
!
Company credits run out — many employees burn their $400/month credits by the 15th because they don't understand cost.
💵 Haiku = Cheapest💰 Sonnet = Balanced💎 Opus = Premium
