Creating Your Own Skill — 3 Ways
From scratch to live capability in under an hour
1
Manual creation — write a skill.md file with domain rules, output format, examples. Use markdown headings and structure.
2
Skills Creator (Claude UI) — describe your need in plain language; Claude scaffolds the complete skill file for you. "Skill creator" is itself a skill!
3
Community / Marketplace — browse repositories like Anthropic's official skills repo or community listings. Download zip, upload, done.
⚠️
Security Warning — Third-Party Skills
Before downloading any community skill, read the skill.md or ask Claude to check it. Malicious skills could try to access or exfiltrate your files.
