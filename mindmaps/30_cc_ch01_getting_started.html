The Three Claude Model Tiers
Match the model to the task — don't use a sledgehammer on a screw
Tier 1 — Fast
⚡
Claude Haiku
Speed First
Fastest and cheapest. Great for repetitive tasks, quick summaries, high-frequency calls where speed beats depth.
Best for: Q&A, drafts, quick lookups
Tier 2 — Balanced
⚖️
Claude Sonnet
Daily Driver
The everyday workhorse. Balances capability and cost — perfect for documents, analysis, and most coding tasks.
Best for: Code, data analysis, writing
Tier 3 — Maximum
👑
Claude Opus
Deep Reasoning
Most powerful. Extended thinking, multi-step plans, complex architecture. Use when depth genuinely matters.
Best for: Architecture, plans, ambiguous tasks

Claude Desktop — Setup & Why It Matters
The bridge between Claude's intelligence and your local environment
1
Download — go to claude.ai/download and install the desktop app for Mac, Windows or Linux
2
Sign in — use your existing claude.ai account or create one; free tier works for getting started
3
MCP support — the desktop app can connect to local MCP servers; the web app cannot do this natively
4
Claude Code bridge — once Claude Code is installed in your terminal, the desktop app can run it as an operator directly
5
Conversation history — your chats persist across sessions, searchable, with project-based memory support

Common Beginner Mistakes
Avoid these to get value from day one
!
Using Opus for everything — 90% of tasks work perfectly on Sonnet; Opus burns credits fast
!
Treating Claude like Google — give it tasks, not questions; "write", "build", "analyse" — not "what is"
!
Stopping at the first response — iterate always; the first output is a draft, not a deliverable
!
Skipping the Desktop app — web-only limits you; Desktop unlocks MCP and Claude Code integration
🚫 Wrong Model 🚫 No Iteration 💡 Task, Not Query

Free vs Paid Plans
When to start free and when upgrading makes sense
🆓
Free plan — access to Sonnet, limited daily messages; enough to learn and experiment
💳
Pro plan — 5× usage, Opus model access, priority during peak hours, longer context
⌨️
Claude Code billing — separate API usage-based billing per token; different from the chat plan
✓
Advice — start free, use Sonnet, upgrade only when you're hitting limits on real daily work
