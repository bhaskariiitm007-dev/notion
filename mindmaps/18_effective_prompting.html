The G.C.A.F. Prompt Framework
Every great prompt has these four elements — the more complete, the better the output
G
Goal
What exactly is the desired outcome? Be specific about the end result you want.
e.g. "Create a short outreach email for US clients"
C
Context
What background does Claude need? Your role, audience, project, constraints.
e.g. "We sell voice agent solutions to medical clinics"
A
Action
What exact steps should Claude take to complete the task? Course of action.
e.g. "Draft, then ask if I want variations"
F
Format
How should the output be structured? Length, bullet points, headers, tone.
e.g. "Short email, formal tone, no more than 100 words"

Custom Instructions — Set Once, Works Forever
Personalize Claude's behavior across every conversation
📋 Ask for Goal, Context & Action
"For any task I provide, please ask me: what is the goal, what context is needed, what action should you take, and what is the desired output format."
❓ Always Ask Clarifying Questions
"Make sure you ask me questions regarding the problem so we can tackle it effectively — don't assume."
🚫 Don't Be a Yes-Man
"Don't always agree with me — make sure you present what is actually best for the task at hand. Act like a senior mentor who corrects my thinking."

Prompting Mistakes to Avoid
Common patterns that lead to shallow, generic responses
✕
Vague questions — "help me with my email" gives you nothing; "write a formal outreach email to a US healthcare startup" gives gold
✕
No context — Claude without context is like a consultant without a brief — they guess and usually miss
✕
Accepting first output — treating Claude as a vending machine instead of a thinking partner you can shape
✕
Asking "questions" not briefing tasks — "What's the best way to...?" vs "Give me a step-by-step plan to..."

Real Iteration Example: Adventure Trip Planning
How layering prompts builds a far better result
1
First prompt: "I'm looking for adventure trip options for 5 people, budget-friendly, North India" — basic output received
2
Iterate: "Make the activities more adventurous" — Claude pivots to paragliding, bouldering, white-water rafting, jungle camping
3
Iterate again: "Add approximate cost estimates per person and for group of 5" — detailed budget breakdown appears
4
Result: Fully usable, specific, costed adventure plan — shaped by three simple follow-up prompts
🏆
The Real Power of Iteration
Each follow-up prompt is a refinement instruction. You're sculpting the output, not starting over. This is how professionals use Claude to get work done.
