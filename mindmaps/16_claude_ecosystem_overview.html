The Three Layers of Claude
One brain — three different ways to access and use it
💬
Claude Chat / Claude.ai
THE BRAIN
The main interface — ask questions, analyze docs, do research, create artifacts. Foundation of everything. Access at claude.ai or desktop app.
💻
Claude Code
THE ENGINEER
Technical coding assistant — Python, JS, full-stack. Gives you code outputs, runs scripts, builds complete applications in a structured coding manner.
🤝
Claude Co-work
THE OPERATOR
Brain + direct functionality. Works inside your machine, connects to tools, has skills and plugins. Handles day-to-day tasks end-to-end as an operator.
💡
⭐ THE FOUNDATION PRINCIPLE
Claude Chat is the main brain — Code and Co-work are built on top of it. Master the chat interface first, and everything else becomes intuitive.

How to Access Claude
Web, desktop, and the free vs paid decision
🌐
Web app — go to claude.ai, sign up with Google or email, works in any browser immediately
💻
Desktop app — downloadable app for Mac/Windows; connects to more local tools and extensions on your machine
🆓
Free plan — full access to core features, limited daily usage — great for starting out
💳
Paid plan — higher usage limits, access to more powerful models (Opus), priority during peak hours
📊
Usage check — go to Settings → Usage to see weekly limits; plan before key meetings to avoid running out
